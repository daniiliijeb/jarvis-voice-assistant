========================================
   JARVIS - ГОЛОСОВОЙ АССИСТЕНТ
========================================

БЫСТРЫЙ СТАРТ
-------------

1. УСТАНОВКА (один раз):
   
   Дважды кликните на файл:
      scripts\setup_env.bat
   
   Дождитесь завершения установки.
   Окно не закроется автоматически - нажмите любую клавишу когда увидите "УСТАНОВКА ЗАВЕРШЕНА!"


2. ЗАПУСК:

   Вариант А - через Python (для разработки):
      .venv\Scripts\python -m jarvis.app.main

   Вариант Б - собрать exe-файл (для обычного использования):
      scripts\build_exe.bat
      
      После сборки запустите:
      dist\Jarvis.exe


3. КАК ПОЛЬЗОВАТЬСЯ:

   - Запустите Jarvis
   - Скажите: "джарвис"
   - Услышите: "Слушаю, сэр"
   - Скажите команду, например: "открой браузер"


ДОСТУПНЫЕ КОМАНДЫ
-----------------

- "открой браузер" - открывает браузер
- "открой папку загрузки" - открывает папку Загрузки


ПРОВЕРКА СИСТЕМЫ
----------------

Чтобы проверить, что всё работает:
   Jarvis.exe --health

Или:
   .venv\Scripts\python -m jarvis.app.main --health


ЛОГИ
----

Все логи сохраняются в папке:
   jarvis\logs\

- jarvis.log - общий лог
- errors.log - только ошибки
- performance.json - метрики производительности (если включено профилирование)


ПРОБЛЕМЫ?
---------

Если микрофон не работает:
   1. Проверьте, что микрофон подключен и включен
   2. Запустите: Jarvis.exe --health
   3. Проверьте драйверы звука в Windows

Если PyAudio не устанавливается:
   Скрипт попробует установить его автоматически через pipwin.
   Если не получилось - проверьте, что у вас установлен Python 3.11.


ТРЕБОВАНИЯ
----------

- Windows 10/11
- Python 3.11 (установите с python.org)
- Микрофон
- Интернет (для распознавания речи через Google)
